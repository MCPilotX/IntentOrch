## Tips for taking screenshots

1. Use a clean browser window (no bookmarks bar, no extensions)
2. Set the browser to light mode for better contrast in README
3. Use real data (e.g., a running 12306-mcp server with a generated workflow)
4. Crop to show only the relevant content area
5. Optimize images with tools like `squoosh` or `tinypng` to keep file size under 500KB each
